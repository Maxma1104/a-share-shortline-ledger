# 股票跟踪数据库

本目录已建立本地 SQLite 数据库，用于结构化记录关注股票、持仓股票、关键价位、每日策略、短线评分和更新日志。

## 文件

| 文件 | 用途 |
| --- | --- |
| `stock_tracking.db` | 实际数据库文件 |
| `db/schema.sql` | 建表结构 |
| `db/seed.sql` | 从当前台账初始化的数据 |
| `股票跟踪台账.md` | 可读版台账，继续保留 |

## 常用查询

查看关注池：

```bash
sqlite3 stock_tracking.db "
SELECT st.code, st.name, m.name AS market, se.name AS sector, w.status, w.key_levels_text, w.invalid_condition
FROM watchlist w
JOIN stocks st ON st.id = w.stock_id
JOIN markets m ON m.id = st.market_id
LEFT JOIN sectors se ON se.id = st.sector_id
ORDER BY w.status DESC, st.code;
"
```

查看当前持仓：

```bash
sqlite3 stock_tracking.db "
SELECT st.code, st.name, h.quantity, h.cost_price, h.last_price, h.unrealized_return_pct, h.stop_loss, h.current_strategy
FROM holdings h
JOIN stocks st ON st.id = h.stock_id
ORDER BY h.position_pct DESC;
"
```

查看短线评分：

```bash
sqlite3 stock_tracking.db "
SELECT sc.score_date, st.code, st.name, sc.stock_type, sc.score, sc.action
FROM shortline_scores sc
JOIN stocks st ON st.id = sc.stock_id
ORDER BY sc.score DESC;
"
```

## 更新原则

1. 新增关注股票：先写入 `stocks`，再写入 `watchlist` 和 `key_levels`。
2. 更新持仓：只改 `holdings`，不要把已持仓股票从风险管理里删除。
3. 新策略或复盘：写入 `daily_strategies`，必要时同步 `update_logs`。
4. 行情快照：写入 `price_snapshots`，并标明数据源和时间。

## RssCast MCP 行情同步

本项目已接入 RssCast MCP，用于获取 A 股实时行情和历史日 K 线。当前落地方式：先把实时行情写入 `price_snapshots`，并同步更新 `holdings.last_price` 和 `holdings.unrealized_return_pct`。

前置条件：

```bash
export RSSCAST_MCP_TOKEN="你的 RssCast MCP Token"
```

预览同步结果，不写数据库：

```bash
python3 scripts/sync_rsscast_prices.py --dry-run
```

同步全部关注和持仓股票：

```bash
python3 scripts/sync_rsscast_prices.py
```

只同步指定股票：

```bash
python3 scripts/sync_rsscast_prices.py --codes 600118 002579
```

分析前刷新并检查数据：

```bash
python3 scripts/refresh_before_analysis.py
```

该命令会连续执行两件事：

1. 刷新 RssCast 实时行情，写入 `price_snapshots` 并更新持仓浮盈。
2. 刷新关注池活跃度，把不活跃股票标记为 `低频观察` 或 `剔除候选`。

只检查当前数据库，不联网刷新：

```bash
python3 scripts/refresh_before_analysis.py --skip-sync
```

只刷新关注池活跃度：

```bash
python3 scripts/refresh_watchlist.py
```

预览关注池刷新结果，不写数据库：

```bash
python3 scripts/refresh_watchlist.py --dry-run
```

同步后检查最近行情快照：

```bash
sqlite3 stock_tracking.db "
SELECT st.code, st.name, ps.snapshot_at, ps.price, ps.change_pct, ps.volume_amount, ps.source
FROM price_snapshots ps
JOIN stocks st ON st.id = ps.stock_id
ORDER BY ps.id DESC
LIMIT 20;
"
```
