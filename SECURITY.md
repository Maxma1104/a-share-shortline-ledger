# Security Policy

This repository is built around a strict privacy boundary.

Private files:

- `stock_tracking.db`
- `db/seed.sql`
- `股票跟踪台账.md`
- `.env`

Public files:

- `db/public_seed.sql`
- `docs/data/watchlist_latest.json`
- `docs/watchlist.md`

If you report a bug, use public seed data or a minimal synthetic example. Do not include brokerage account data, access tokens, or personal trading records.
