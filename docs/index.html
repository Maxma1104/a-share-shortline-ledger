这个项目怎么用

1. 直接查公开关注池

普通用户无需配置 Token，打开本页即可查看每日 18:00 更新的公开关注池。

2. 本地跑完整台账

维护者可使用 SQLite 保存真实台账，并在分析前运行刷新脚本确认数据新鲜度。

3. 只公开脱敏结果

公开导出不包含持仓数量、成本价、仓位比例和私人复盘，只展示可查验的关注池状态。
